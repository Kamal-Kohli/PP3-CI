## Overview
Welcome to the Battleship Game! This repository houses a classic naval strategy game implemented in [Python programming language](https://github.com/Kamal-Kohli/naval-war-pp3/blob/main/run.py), which runs in the Code Institute mock terminal on Heroku app. Battle it out on the high seas, strategize your moves, and sink your opponent's ships. Challenge your friends or test your strategic skills against a smart AI opponent in this engaging and interactive Battleship experience.
